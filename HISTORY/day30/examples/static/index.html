Airi-Cpp-Server-Lib

高性能 C++ Reactor HTTP 服务器 · day30 生产就绪演示

epoll/kqueue multi-reactor middleware rate-limit auth CORS gzip sendfile prometheus

开放端点（无需鉴权）

  • GET /health健康检查（K8s liveness/readiness）
  • GET /metricsPrometheus 文本格式指标
  • GET /static/*静态资源（ETag / 304 / Range）
  • GET /files/*用户文件（sendfile 零拷贝）

受保护端点（需 Bearer 或 API Key）

  • GET /api/usersJSON 用户列表（直接点击会 403）
  • OPTIONS /api/usersCORS 预检 → 204

调用示例

  • curl -H "Authorization: Bearer demo-token-2024" http://localhost:8888/api/users
  • curl -H "X-API-Key: api-key-001" http://localhost:8888/api/users
  • for i in $(seq 1 300); do curl -s -o /dev/null -w "%{http_code}\n" http://localhost:8888/; done | sort | uniq -c
